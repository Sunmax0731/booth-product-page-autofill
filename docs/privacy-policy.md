# Privacy Policy

Effective date: 2026-04-26

BOOTH Product Page Autofill is a Chrome extension for filling BOOTH item edit pages from a user-selected Markdown file and user-selected image files.

## Data Collection

This extension does not collect, transmit, sell, or share personal information.

The extension processes the following information locally in the browser:

- Markdown files selected or dropped by the user
- Image files selected or dropped by the user
- Form fields on `https://manage.booth.pm/items/*/edit`

This information is used only to fill the currently open BOOTH item edit page.

## Data Storage

The extension does not store uploaded Markdown or image files on an external server.

The extension does not use analytics, tracking, advertising, or remote logging.

## Network Requests

The extension does not send user files or BOOTH page contents to third-party servers.

## Permissions

The extension uses these Chrome permissions:

- `activeTab`: to run only on the currently active BOOTH edit page after the user starts autofill.
- `scripting`: to inject the autofill script into the current BOOTH edit page.
- `https://manage.booth.pm/items/*/edit`: to limit the extension to BOOTH item edit pages.
